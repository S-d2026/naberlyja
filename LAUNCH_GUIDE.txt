STEP 1 — SET UP SUPABASE DATABASE (15 minutes)
=================================================================

1. Go to supabase.com and sign in
2. Open your existing project (or create new → name it "naberly-ja")
3. Click "SQL Editor" in the left sidebar
4. Click "New Query"
5. Open the file: supabase/schema.sql
6. Copy ALL the contents
7. Paste into Supabase SQL Editor
8. Click "Run" (green button)
9. You should see "Success. No rows returned"

That creates:
- parishes table (all 14 Jamaica parishes)
- profiles table (user data)
- listings table (all posts)
- impact_stories table (community pulse)
- saved_listings table (favorites)
- boosts table (payment requests)
- All security rules

=================================================================
STEP 2 — GET YOUR SUPABASE KEYS (5 minutes)
=================================================================

1. In Supabase, click "Settings" (gear icon, left sidebar)
2. Click "API"
3. You will see:
   - Project URL → copy this
   - anon public key → copy this
   - service_role key → copy this (keep secret!)

=================================================================
STEP 3 — SET UP YOUR PROJECT IN VS CODE (10 minutes)
=================================================================

1. Open VS Code
2. Open your existing naberlyja folder
   (the one already deployed to Vercel)

3. Create a new file called .env.local in the root folder
   (same level as package.json)

4. Paste this into .env.local and fill in YOUR values:

NEXT_PUBLIC_SUPABASE_URL=https://YOUR_PROJECT_ID.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=paste_your_anon_key_here
SUPABASE_SERVICE_ROLE_KEY=paste_your_service_role_key_here
NEXT_PUBLIC_SITE_URL=https://naberlyja.com
NEXT_PUBLIC_NABERLY_WHATSAPP=+19174432797

5. Save the file

IMPORTANT: Never share .env.local with anyone.
IMPORTANT: Make sure .env.local is in your .gitignore file.

STEP 4 — INSTALL SUPABASE PACKAGE (2 minutes)
=================================================================

1. In VS Code, open Terminal (top menu → Terminal → New Terminal)
2. Type this command and press Enter:

3. Wait for it to finish (you'll see "added X packages")
