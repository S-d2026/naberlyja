# Jamaica Payment Rollout

## Best launch setup
### For marketplace sellers
- WhatsApp order
- call order
- cash on meetup
- cash on delivery
- LynkBiz QR / request links

### For Naberly JA platform paid features
- WiPay hosted links / checkout
- later: alternate card processor only if needed

## Why this is better
Many local sellers will convert better with:
- WhatsApp
- direct contact
- simple payment request
than a forced full cart checkout on day one.
