This is the final combined repo handoff for deployment.

## Stack
- Next.js
- Vercel
- Supabase Auth
- Supabase Postgres
- Supabase Storage
- Supabase Realtime

## Jamaica-first payment recommendation
Use these in this order:
1. WhatsApp + call + cash / meetup for community sellers
2. WiPay hosted links / checkout for featured listing payments
3. LynkBiz for simple QR / payment requests
4. deeper payment flows later, after launch traction
